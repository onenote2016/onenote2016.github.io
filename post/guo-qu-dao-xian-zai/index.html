有时候，前人做吃过茸毛饮血，我们不应该再吃了，而且，不能再浑浑噩噩下去了

从考研开始，我感觉我学到了很多，从谦逊，克己和步步为营到
谦逊，开明和平和，后者继承前者。
从，英语阅读强化开始，唐迟老师，讲的比较好，我认为，
他是可以和汤家凤可以做对比的，他教给我常常去总结。
在这段时间，我醒悟了：人是可以害怕的，
但是，不能让害怕控制了你，害怕让我谨慎去面对。

温习中国近代史，翻过一篇又一篇文章，不免生气；
为什么每一代人都要都得自己吃一回蜘蛛，
吃的满嘴黑毛绿血，
才明白蜘蛛不好吃。
如果总是重复前人的痛苦，重复前人的努力，
整个民族智慧指挥消耗在一代又一代的重复中。
                                                                ————龙应台

• 害怕可以让我感觉到恐慌，同时，我更加主动小心谨慎去做事情
• 失败，不仅仅意味着你输了，
    而是要让你要反思你的失误点在哪里，相反你可以获得到你没有东西，
    而不是停留在失败的阴影中
• 做事和学习一样，要注重效率和质量，而不是速度
• 从之前的针锋相对，到现在的善良开明
• 每天，要知道，我获得了什么，失误了什么
• 不要拖延

自知

  1. 缺点

    • 专业技能不过硬
    • 电路知识了解太少
    • 不独立
    • 不爱说话
    • 不主动了解信息

  2. 优点

    • 有毅力
    • 自律
    • 善良和针锋相对
    • 喜欢安静

  3. 能做到坚持的人，有两种:

    • 一种是有强大信念的人；
    • 另一种是能沉下心的人。

  4. 做到坚持

    • 做一个目标为导向的人
    • 重复是培养习惯的开始
    • 监控自己的学习过程
    • 量力而行

Note:

要想到就做到。
对待坚持，我们首先要让自己有决心，要相信自己能够成功，
其次是把努力当成一种习惯，转变对困难的态度。
不要觉得一件事情好难，我们就放弃，
我们要时刻鼓励自己，让自己能够坚持下来。
既然知道人生珍贵，那请你们不要随意放弃自己，
不要随意否定自己，考上什么学校不重要，考上你想要的学校才重要，
请大家自信一点，从容一点，你要记得你是自己的英雄！
